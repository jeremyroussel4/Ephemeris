## ✨ About

Ephemeris is a lightweight JavaScript project for the iOS application **Scriptable**.

Each time the script runs, it randomly selects a sentence from a JSON collection and displays it as a daily reflection.

The goal is to create a quiet moment of attention through simple, carefully chosen words.

---

## 📖 Example

> Different sizes are not necessarily a bug.
  Des tailles différentes ne sont pas forcément un bug

## 🚀 Features

- 🎲 Random quote generation
- 📄 JSON-based sentence collection
- ⚡ Lightweight and fast
- 📱 Compatible with Scriptable
- ✍️ Easy to expand with new reflections

## 💻 Installation

1. Install **Scriptable** on iPhone or iPad.
2. Clone this repository.
3. Copy `ephemeris.js` into Scriptable.
4. Add or edit the `phrases.json` file.
5. Run the script.
